📦 Inventory Scanner

To auto-create your Google Sheet and connect everything, you need a Google Cloud Client ID. This is a one-time setup that takes about 10 minutes.

Get your Client ID

1
Go to console.cloud.google.com → create a new project (name it anything)
2
Search "APIs & Services" → Enable APIs → enable these three: Google Sheets API, Google Drive API, Apps Script API
3
Go to Credentials → Create Credentials → OAuth Client ID → type: Web application
4
Under "Authorized JavaScript origins" add: https://xwxexixrxdxox-spec.github.io → click Create
5
Copy the Client ID (ends in .apps.googleusercontent.com) and paste it below
