Get your Client ID

1
Go to console.cloud.google.com → create a new project
2
APIs & Services → Library → enable: Google Sheets API, Google Drive API
3
Credentials → Create Credentials → OAuth Client ID → type: Web application
4
Under "Authorized JavaScript origins" add: https://xwxexixrxdxox-spec.github.io
5
Copy the Client ID (ends in .apps.googleusercontent.com) and paste below
